# 256 Circle Project

## Description
A React application built with Vite and TypeScript.

## Prerequisites
- Node.js (v18+)
- npm

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server

## Deployment
Deployed on Vercel: https://256circle-2e8f5pxcb-pao7os-projects.vercel.app

## Technologies
- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- Zustand
